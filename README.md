# Calculator Application

A simple calculator application developed using HTML, CSS, and JavaScript. 
This project performs basic arithmetic operations with an interactive user interface.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Delete button
- Responsive design

## Project Purpose

The purpose of this project is to understand basic frontend development concepts, user interaction, and JavaScript functionality.

## How to Run

1. Download or clone the repository
2. Open `index.html` in a browser
3. Use the calculator
